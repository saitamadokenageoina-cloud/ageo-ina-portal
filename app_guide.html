このアプリの使い方

組合員さんが迷わず使えるよう、目的別に操作方法をまとめました。

このポータルとは？

  1. 埼玉土建一般労働組合 上尾伊奈支部の組合員向けサポートアプリです。
  2. スマートフォンのブラウザで開くだけで使えます。アプリストアからのインストールは不要です。
  3. 現場の安全確認、熱中症アラート、36協定、手続き書類、資料本棚などをまとめて確認できます。
公開URL：https://saitamadokenageoina-cloud.github.io/ageo-ina-portal/

ホーム画面の使い方

区分できること
現場で使う熱中症AIアラート、安全チェック、36協定、DOKENギルドを開きます。
組合員サービスどけんカード、加入メリット診断、CCUS能力評価サポート、建設業許可チェッカーを使います。
手続き・情報必要書類、加入メリット、計算ツール、講習日程、資料本棚、YouTube、支部カレンダーを確認します。
迷ったときは、目的に近いカードをタップしてください。各ページの左上「戻る」から前の画面へ戻れます。

熱中症AIアラート

  1. ホーム画面の「熱中症AIアラート」をタップします。
  2. 位置情報の許可が出たら「許可」を選びます。
  3. 現在地の気象情報から危険度を確認します。
位置情報を拒否した場合は、ブラウザや端末設定から位置情報の許可を戻してください。

現場安全チェックリスト

  1. 現場名、点検者、日付を入力します。
  2. 該当する安全項目にチェックを入れます。
  3. プレビューで確認してから印刷またはPDF保存します。

手続き・必要書類ガイド

  1. 「手続き・情報」から「手続き・必要書類ガイド」を開きます。
  2. 国保、共済、加入、労働保険など目的に近い項目を選びます。
  3. 必要書類と問い合わせ先を確認します。

資料本棚

  1. 「資料本棚」を開き、読みたい資料を選びます。
  2. デジタルブックで読みづらい場合は「PDF表示」を選びます。
  3. 端末標準のPDFビューアで拡大・検索・共有ができます。

印刷・PDF保存の基本

  1. 36協定や安全チェックで必要事項を入力します。
  2. 「プレビューを表示」をタップして内容を確認します。
  3. 「印刷・PDF保存」をタップします。
  4. 印刷画面でプリンターを選ぶか、「PDFに保存」を選びます。
iPhoneはSafari、AndroidはChromeでの利用がおすすめです。

iPhoneでPDF保存する

  1. Safariで対象ページを開きます。
  2. 印刷プレビューを開きます。
  3. プレビュー画面を2本指で広げる操作をします。
  4. 共有ボタンから「ファイルに保存」を選びます。

AndroidでPDF保存する

  1. Chromeで対象ページを開きます。
  2. 印刷画面を開きます。
  3. 送信先を「PDF形式で保存」に変更します。
  4. 保存先を選び、保存します。

ホーム画面に追加する

iPhone / Safari

  1. Safariで公開URLを開きます。
  2. 画面下の共有ボタンをタップします。
  3. 「ホーム画面に追加」を選びます。
  4. 名前を確認して「追加」をタップします。

Android / Chrome

  1. Chromeで公開URLを開きます。
  2. 右上のメニューをタップします。
  3. 「アプリをインストール」または「ホーム画面に追加」を選びます。
  4. 確認画面で「追加」または「インストール」をタップします。

DOKENギルドのクラウド保存・メール通知を有効にする

GitHub Pagesだけでは、投稿データの保存やメール送信はできません。支部のGoogleアカウントでGoogle Apps Scriptを発行し、そのURLを設定します。
  1. 支部のGoogleアカウントでGoogleスプレッドシートを新規作成します。名前は「DOKENギルド投稿管理」などにします。
  2. スプレッドシートURLの /d/ と /edit の間にある長い文字列を控えます。これがスプレッドシートIDです。
  3. スプレッドシートの「拡張機能」から「Apps Script」を開きます。
  4. このリポジトリの docs/guild-apps-script-template.js の内容をApps Scriptに貼り付けます。
  5. Apps Scriptの「プロジェクトの設定」からスクリプトプロパティを追加します。
SPREADSHEET_ID = 手順2で控えたスプレッドシートID ADMIN_PIN = 投稿削除に使う管理者PIN
  1. 「デプロイ」→「新しいデプロイ」を選びます。
  2. 種類は「ウェブアプリ」、実行ユーザーは「自分」、アクセスできるユーザーは「全員」にします。
  3. 初回はGoogleの承認画面が出ます。支部アカウントで許可します。
  4. 発行されたウェブアプリURLをコピーします。
  5. assets/js/guild-config.js の apiUrl にURLを入れて、GitHubへpushします。
window.DOKEN_GUILD_CONFIG = { apiUrl: 'https://script.google.com/macros/s/発行されたURL/exec', notifyEmail: 'ageoina@saitama-doken.or.jp' };
設定後は、全員が同じ投稿を見られます。新しい投稿があると支部メール ageoina@saitama-doken.or.jp に通知されます。

管理者として投稿を削除する

  1. DOKENギルド画面で「管理者」をタップします。
  2. Apps Scriptに設定した ADMIN_PIN を入力します。
  3. 各投稿に表示される「削除」ボタンから不要な投稿を削除します。
管理者PINは支部内の管理者だけで共有してください。公開ページや資料には載せないでください。

古い画面が表示される

  1. 画面を下に引っ張って更新します。
  2. 改善しない場合はブラウザの履歴・キャッシュを削除します。
  3. ホーム画面に追加したアイコンから開いている場合は、一度ブラウザから直接URLを開き直します。

位置情報が取得できない

  1. 端末の設定でSafariまたはChromeの位置情報を許可します。
  2. ブラウザでこのサイトの位置情報許可をオンにします。
  3. ページを開き直して再取得します。

DOKENギルドが「ローカル表示中」になる

  1. assets/js/guild-config.js の apiUrl が空欄でないか確認します。
  2. Apps ScriptのデプロイURLが /exec で終わっているか確認します。
  3. Apps Scriptのアクセス権限が「全員」になっているか確認します。
  4. スクリプトプロパティの SPREADSHEET_ID が正しいか確認します。
